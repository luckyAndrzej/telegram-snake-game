# Обновление системы вывода средств

## Изменения в server.js

Добавлена защита от частых запросов (30 секунд) и интеграция с TON SDK.

### Необходимые шаги для настройки:

1. Установить библиотеки TON:
```bash
npm install @ton/ton @ton/crypto
```

2. Добавить в `.env`:
```
ADMIN_SEED=word1 word2 word3 ... word24
```

3. `ADMIN_SEED` - это seed-фраза администраторского кошелька (24 слова), из которого будут отправляться выплаты.

## Логика вывода:

- Защита от частых запросов: 30 секунд между запросами
- Конвертация: 1 USDT = 0.5 TON (можно настроить через курс)
- TON транзакция отправляется через TonClient
- После успешной транзакции баланс обновляется
- Хеш транзакции отправляется клиенту
